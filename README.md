# Chic Clinic — nettside

Nettside for Chic Clinic AS (Trondheim), bygget med Next.js.

## Kom i gang

```bash
npm install
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000).

## Bygg

```bash
npm run build
```

## Deploy

Hosting på Vercel. Deploy fra prosjektroten med Vercel CLI (se `context/DEV.md`).

## Innhold

Bedriftsinformasjon, kontaktfelt og SEO-grunnlag vedlikeholdes primært i `src/lib/site-config.ts`.
